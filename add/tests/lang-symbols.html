Language and Symbols

How do screen readers read symbols?

Symbols with paragraph tags around each.

!

@

#

$

%

VO reads all symbols. JAWS reads the ! as "exclaim."

Symbols with empty space around each: ! @ # $ %

VO reads all symbols except the exclamation mark.

Symbols with empty space around each and exclamation mark in second place: @ ! # $ %

Symbols with a span tag around each: ! @ # $ %

VO reads all symbols except the exclamation mark.

With an aria label: !

With an span title: !

With an abbr title: !

As an HTML entity: !

As a paragraph:

!

VO reads all symbols except the exclamation mark.
