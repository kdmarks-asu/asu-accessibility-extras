Images: Inline SVG

Tests

  • JAWS ignores SVG/role=img with an empty alt tag. Removing those examples.
  • JAWS ignores SVG/role=img without an alt tag. Removing those examples.

  1. Title (and desc)

    The W3C recommends using a <title> for an SVG's alternative text but doesn't mention adding a role="img"

    JAWS reads title tag but not desc
    VoiceOver reads the title and desc but says we're in an empty group. Doesn't recognize this as an image, so can't use G shortcut.


    <svg xmlns=http://www.w3.org/2000/svg>
   <title>This is the title tag</title>
   <desc>This is the description tag</desc>
   <circle cy="60" r="55" stroke="black" stroke-width="2" fill="red" />
</svg>

  2. Title (and desc) with aria-labelledby

    JAWS reads title and desc
    VoiceOver reads the title and desc but says we're in a group


    <svg aria-labelledby="title1 description1" xmlns=http://www.w3.org/2000/svg>
   <title id="title1">This is the title tag</title>
   <desc id="description1">This is the description tag</desc>
